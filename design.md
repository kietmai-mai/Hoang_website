# Team MAI Mortgage Website - Design Style Guide

## Design Philosophy

### Color Palette
- **Primary**: Royal Purple (#6B46C1) - Luxury, trust, sophistication
- **Secondary**: Warm Cream (#F7F3E9) - Elegance, approachability
- **Accent**: Champagne Gold (#D4AF37) - Premium, success, achievement
- **Text**: Charcoal (#2D3748) - Professional, readable
- **Background**: Soft Ivory (#FEFEFE) - Clean, spacious

### Typography
- **Display Font**: Playfair Display (serif) - Elegant headlines, luxury feel
- **Body Font**: Inter (sans-serif) - Clean, modern readability
- **Accent Font**: Crimson Text (serif) - Sophisticated details

### Visual Language
- **Luxury Editorial**: Inspired by high-end financial publications
- **Sophisticated Minimalism**: Clean layouts with purposeful white space
- **Professional Warmth**: Balancing corporate credibility with personal approachability
- **Bilingual Elegance**: Seamless integration of English and Vietnamese content

## Visual Effects & Styling

### Used Libraries & Effects
1. **Anime.js**: Smooth number animations for statistics, gentle fade-ins
2. **Typed.js**: Typewriter effect for hero headlines
3. **Splide.js**: Elegant carousels for success stories and property showcases
4. **ECharts.js**: Clean data visualization for calculator results
5. **p5.js**: Subtle particle background effects for hero section
6. **Splitting.js**: Letter-by-letter text animations for section headers
7. **Pixi.js**: Smooth image transitions and hover effects

### Header & Hero Effects
- **Gradient Text Animation**: Royal purple to gold color cycling on main headline
- **Subtle Parallax**: Gentle background movement on scroll
- **Professional Portrait Integration**: MAI hero image with elegant overlay text
- **Soft Particle Background**: Minimal floating elements in royal purple tones

### Interactive Elements
- **3D Tile Hover**: Loan program segments lift and tilt on hover
- **Ken Burns Effect**: Subtle zoom and pan on success story images
- **Smooth Transitions**: All page navigation and interactions use easing curves
- **Calculator Animations**: Real-time number morphing with smooth transitions

### Scroll Motion Design
- **Gentle Reveals**: Content fades in with 20px upward movement
- **Staggered Animations**: Cards and tiles appear with 100ms delays
- **Progress Indicators**: Subtle scroll progress for long pages
- **Parallax Restraint**: Maximum 8% translateY for decorative elements

### Image Treatment
- **Professional Photography**: High-quality, well-lit business and lifestyle images
- **Consistent Filtering**: Warm color grading with slight purple tint
- **Hover Overlays**: Elegant gradients on interactive elements
- **Aspect Ratios**: Consistent 16:9 for hero, 4:3 for cards, 1:1 for avatars

### Background Design
- **Consistent Base**: Soft ivory background throughout all pages
- **Decorative Elements**: Subtle geometric patterns in light purple
- **Section Differentiation**: Gentle cream tints for content areas
- **No Gradients**: Solid colors only, maintaining sophisticated aesthetic

### Mobile Optimization
- **Touch-First Design**: Large, accessible touch targets
- **Simplified Animations**: Reduced motion for mobile performance
- **Readable Typography**: Optimized font sizes for small screens
- **Thumb-Friendly Navigation**: Bottom-aligned primary actions

## Accessibility & Performance
- **High Contrast**: 4.5:1 minimum ratio for all text
- **Focus Indicators**: Clear purple outlines for keyboard navigation
- **Reduced Motion**: Respects user preferences for motion sensitivity
- **Fast Loading**: Optimized images and progressive enhancement
- **Screen Reader Support**: Semantic HTML and ARIA labels

## Brand Integration
- **Team MAI Identity**: Professional yet approachable aesthetic
- **Bilingual Support**: Visual design accommodates both English and Vietnamese text
- **Trust Indicators**: Subtle integration of awards and certifications
- **Social Proof**: Elegant display of client success stories and testimonials
